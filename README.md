## Tech Stack

**Client:** Express-handlebars, TailwindCSS, Jquery

**Server:** Node, Express, MongoDB


## Run Locally

Clone the project

```bash
  git clone https://github.com/nhatloi1204/SOA-Final.git
```

Go to the project directory

```bash
  cd SOA
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run server
```
